How to contribute

  1. Fork this project.
  2. Add new posts to the _posts folder with properly formed YAML up top. At minimum: 
              ---
          Layout: post
          Title: YourTitleHere
          ---
  3. Feel free to change or add layouts, css, etc.
  4. Add yourself as a contributor to the homepage (does someone know how to break these out into, say, a separate _contributors folder?)
  5. Commit your changes.
  6. Submit a pull request
