How to contribute

  1. Fork this project. The code lives here.
  2. Add new posts to the _posts folder with properly formed YAML up top. At minimum: 
            
---
layout: post
title: yourtitlehere
month: May
day: 20
year: 2014
post_author: yournamehere
---
          ---
  3. Feel free to change or add layouts, css, etc.
  4. Add yourself as a contributor to the homepage. There are lots of cool ways to handle authors that I don't have time to implement just yet - but you can.
  5. When you're satisfied and it works, commit your changes and Submit a pull request
